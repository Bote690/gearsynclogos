# Official logo/source candidates

This file tracks official manufacturer media/press/brand sources to review before adding full-color transparent logos to the public catalog.

Rules:

- Prefer official media kits, press rooms, brand portals, or downloadable assets.
- Add files to `logos/` only after checking that the asset is full-color/approved and usable on transparent or dark backgrounds.
- Record the source URL and usage note in `logos.json`.
- Avoid third-party shops, articles, search result thumbnails, or monochrome-only icon sets.

## Next import workflow

1. Open the official source for a brand.
2. Locate a current full-color logo/brand asset or press kit.
3. Download only assets suitable for transparent-background display.
4. Add the file to `logos/` as PNG/SVG/WebP.
5. Add an entry to `logos.json` with `sourceUrl`, `usage`, and `status`.
6. Test in GearSync app via the Catalog button.
